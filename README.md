## Консольное приложение для управления книгами в библиотеке

### Запуск

```
# Сборка
mvn package

# Запуск
java -jar target/library.jar

# Тесты
mvn test
```

### Описание

Проект разбит на 4 класса:  

- `Main` - точка входа в приложение;
- `LibraryService` - бизнес-логика (добавление, удаление, поиск, статистика);  
- `CommandParser` - разбор ввода и формирование ответа;
- `Book` - модель книги;

Книги хранятся в `LinkedHashMap<Integer, Book>`. HashMap даёт доступ по ID за O(1), а Linked сохраняет порядок вставки, что нужно для вывода списка без сортировки
